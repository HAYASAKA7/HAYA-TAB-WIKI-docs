# Quick Start Guide

  Get up and running with HAYA-TAB in 5 minutes.

  ## Step 1: Launch the App

  After [installing](Installation), launch HAYA-TAB. You'll see an empty library grid.

  ## Step 2: Add Your First Tab

  **Option A: Upload a Tab**
  1. Right-click on empty space in the library
  2. Select **"Upload TAB"**
  3. Choose a PDF, Guitar Pro, or MusicXML file
  4. The file is copied to HAYA-TAB's internal storage

  **Option B: Link an Existing File**
  1. Right-click → **"Link Local TAB"**
  2. Select your file
  3. The file stays in its original location (HAYA-TAB creates a reference)

  ## Step 3: Organize with Categories

  1. Right-click → **"New Category"**
  2. Name it (e.g., "Rock", "Jazz", "Practice")
  3. Drag tabs into categories to organize them

  ## Step 4: Set Up Folder Sync (Optional)

  Automatically import tabs from a folder:

  1. Click the **⚙️ Settings** icon (bottom-left)
  2. Go to **"Sync Paths"** section
  3. Click **"Add Path"** and select your tabs folder
  4. Click **"Sync Now"** to import existing files
  5. New files added to this folder will be auto-imported

  ## Step 5: Open a Tab

  - **Double-click** a tab to open with your system's default app
  - **Right-click → "Open with Inner Viewer"** to use the built-in viewer

  ## Step 6: Configure WebDAV (Optional)

  Access tabs from your cloud storage:

  1. Settings → **WebDAV** section
  2. Enter your server URL (e.g., `https://cloud.example.com/remote.php/dav/files/username/`)
  3. Enter username and password
  4. Click **"Test Connection"**
  5. Browse your cloud library from the sidebar

  ## Keyboard Shortcuts

  | Action | Default Key |
  |--------|-------------|
  | Play/Pause (GP Viewer) | `Space` |
  | Toggle Loop | `L` |
  | Jump to Start | `I` |
  | Toggle Auto-Scroll (PDF) | `N` |
  | Speed Up | `.` |
  | Speed Down | `,` |

  Customize shortcuts in **Settings → Key Bindings**.
