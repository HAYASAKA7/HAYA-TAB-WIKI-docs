# PDF Viewer

  HAYA-TAB includes a built-in PDF viewer powered by PDF.js.

  ## Opening PDFs

  1. Right-click a PDF tab → **"Open with Inner Viewer"**
  2. Or double-click if PDF viewer is set as default

  ## Controls

  | Control | Action |
  |---------|--------|
  | Scroll | Navigate pages |
  | `+` / `-` | Zoom in/out |
  | Fit Width | Auto-fit to window width |
  | Page Input | Jump to specific page |

  ---

  ## Auto-Scroll Feature

  Perfect for hands-free practice!

  ### Enable Auto-Scroll
  - Press **`N`** (default) or click the auto-scroll button in toolbar
  - The page scrolls automatically at a constant speed

  ### Adjust Speed
  - Press **`.`** to increase speed
  - Press **`,`** to decrease speed
  - Or use the speed input field in the toolbar

  ### Stop Auto-Scroll
  - Press **`N`** again
  - Or click the auto-scroll button
  - Automatically stops at end of document

  ---

  ## Keyboard Shortcuts

  | Action | Default Key | Customizable |
  |--------|-------------|--------------|
  | Toggle Auto-Scroll | `N` | ✅ |
  | Speed Up | `.` | ✅ |
  | Speed Down | `,` | ✅ |
  | Zoom In | `Ctrl/Cmd + =` | ❌ |
  | Zoom Out | `Ctrl/Cmd + -` | ❌ |

  Customize shortcuts in **Settings → Key Bindings**.

  ---

  ## Tips

  - **Large PDFs:** The viewer handles large files efficiently with lazy page loading
  - **Print:** Use your browser's print function (`Ctrl/Cmd + P`)
  - **External App:** Right-click → "Open with Default App" to use your system PDF reader
