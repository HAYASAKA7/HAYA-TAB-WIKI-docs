# HAYA-TAB

  Welcome to the official wiki for **HAYA-TAB** - a lightweight music tab manager for guitarists and musicians.

## What is HAYA-TAB?

  HAYA-TAB helps you organize, search, and play your PDF and Guitar Pro tabs in one unified application. Built with Go and Vue 3, it
   offers a modern desktop experience with powerful features.

## Key Features

- 📁 **Tab Management** - Upload or link PDF, Guitar Pro (.gp, .gp3, .gp4, .gp5, .gpx), and MusicXML (.xml, .musicxml, .mxl) files
- 🔍 **Instant Search** - Full-text search with fuzzy matching across titles, artists, and albums
- ☁️ **Cloud Sync** - WebDAV integration for Nextcloud, ownCloud, and other cloud services
- 🎸 **Built-in Viewers** - PDF viewer with auto-scroll, Guitar Pro viewer with playback controls
- 🌐 **Multi-language** - English, Chinese (Simplified/Traditional), Japanese with auto-detect system language
- 🔒 **Security** - XSS prevention and machine-specific encryption for credentials
